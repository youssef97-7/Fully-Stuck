# About Fully Stuck

## Project Vision

**Fully Stuck** is a comprehensive book management and reading tracking platform designed to help book enthusiasts organize, discover, and track their reading journey. Whether you're a casual reader or a book collector, Fully Stuck provides the tools you need to manage your personal library effectively.

## What We Do

Fully Stuck is a web-based application that allows users to:

- **Organize Your Library**: Maintain a digital catalog of books you own, have read, or want to read
- **Track Reading Status**: Categorize books into multiple statuses including wishlist, current reading, completed reads, borrowed items, and personal favorites
- **Manage Book Information**: Store and access detailed information about each book including author, category, ratings, and descriptions
- **Stay Connected**: Receive notifications about your reading activities and library updates

## Who Is It For?

- **Book Lovers**: Anyone passionate about books who wants to organize and track their reading
- **Avid Readers**: Readers who maintain large personal libraries and want to manage them digitally
- **Book Clubs**: Communities that want to track shared books and reading progress
- **Libraries**: Organizations looking to manage their book inventory and user reading habits

## Core Features

### Reading Status Management
Organize books by reading status:
- **Wishlist**: Books you want to read in the future
- **Currently Reading**: Books you're actively reading
- **Read**: Books you've completed
- **Borrowed**: Books borrowed from others
- **Favorites**: Books you love and want to remember

### Comprehensive Book Catalog
Every book includes:
- Title and description
- Author information
- Category classification
- Rating system (0-5 stars)
- Cover images for visual organization
- Availability tracking

### User Profiles
Create personalized accounts with:
- Custom user profiles
- Personal reading preferences
- Library organization
- Notification preferences

### Notification System
Stay updated with:
- Reading reminders
- Book availability updates
- Library notifications
- Community updates

## Technology

Built with modern web technologies:
- **Django**: A robust Python web framework
- **SQLite**: Reliable database management
- **Responsive Design**: Works on desktop and mobile devices
- **Image Processing**: Professional cover image handling
